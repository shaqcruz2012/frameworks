using bootstrap

  • To center a div, use class=justify-content-center align-items-center
  • To include padding in a class in bootstrap, use p-3 where 3 is the number of pixels
  • The container grid is based on 12 columns
    • class=col-sm-4 sets that class to take up 4/12 available columns when viewed in small screens
    • class=col-md-8 sets that class to take up 8/12 available columns when viewed on medium screens
    • class=col-lg-2 sets that class to take up 2/12 available columns when viewed on large screens
    • class=col-sm-12 col-md-6 col-lg-2 sets the class for each space
